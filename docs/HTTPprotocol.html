HTTP & HTTPS PROTOCOL

  • _____

    HTTP stands for Hyper Text Transfer Protocol. It is the language that one computer uses to communicate with another computer when requesting a document. In particular, it is the language between web browsers and servers. An examples is when you want to access a website such as youtube.com. My computer will submit a get request to the youtube server asking for the code for the youtube page.
