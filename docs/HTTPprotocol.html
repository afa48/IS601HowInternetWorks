HTTP & HTTPS PROTOCOLS

  • HTTP stands for Hyper Text Transfer Protocol and HTTPS stands for Hyper Text Transfer Protocol Secure. HTTP is the language that one computer uses to communicate with another computer when requesting a document. In particular, it is the language between web browsers and servers. An examples is when you want to access a website such as youtube.com. My computer will submit a get request to the youtube server asking for the code for the youtube page. HTTPS is a secure protocol that uses SSL and TLS. SSL, or Secure Sockets Layer and TLS, or Transport Layer Security are cryptographic protocols that authenticate data transfer between servers. The HTTPS connection is indicated by a small lock in the browser bar next to https. The way it works is when a website asks a browser to be a part of a secure connection, it first provides a digital certificate. This indicates that it is officially the website it is claiming to be. Certificate Authorities issue these certificates. If a website does not have a secure connection, a warning sign will appear on your screen. Examples of HTTP are websites like internet forums or educational sites. HTTPS include sites such as banking websites and shopping websites.
