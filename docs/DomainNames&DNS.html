Domain Names & DNS

  • _____

    A Domain Name is basically like the address to your website. It consists of two parts, the websites name and the domain name extension. For facebook.com - the domain name is facebook and the extension is .com. If there was no domain name, the full IP address of the website would have to be entered which is very difficult for people to memorize.

  • DNS is also known as the Domain Name System. This system is like a phonebook of the Internet. It is a database where domain names correspond with an IP address. The computer uses the Domain Name System by translating domain names that are readable by humans to IP addresses that are readable by computers. The computer looks up the domain name and obtains the associated IP address. The computer then connects to the destination on the internet. For example, if you want to go to the page google.com, the computer will look up the IP address for that domain name and connect you to google.com. ICANN manages domain names by coordinating the distribution of IP addresses so that there is no repetition of addresses. ICANN is the central database for all of the IP addresses.
